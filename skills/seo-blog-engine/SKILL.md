# Programmatic SEO & Content Marketing Skill

Governance engine for automated generation of deep, search-intent optimized, ASCI-compliant brand crisis case studies.

## Skill Frontmatter Metadata
- **name**: seo-blog-engine
- **description**: Automated content generation system producing highly structured, SEO-optimized corporate compliance, linguistic risk, and PR controversy case studies for the BrandCheck Pro journal.

## Core Directives

### 1. Architectural Structure
All articles generated by this engine must reside in `blog/` using clean URL slugs: `/blog/brandname-campaign-crisis-year.html`.
Every article must inherit the global theme initialization block and visual styling (Outfit for titles, Inter for copy, JetBrains Mono for metrics).

### 2. Search Engine Optimization (SEO)
- **Crawlability**: Inject JSON-LD `TechArticle` schema directly into the `<head>` of each file.
- **Intent Target**: Long-tail keyword matching for brand reputation queries (e.g., "[Brand Name] ad controversy", "ASCI ruling [Campaign Name]").
- **Hierarchy**: Strict `<h1>` for page title, `<h2>` for sections, and `<h3>` for scorecard elements.

### 3. Visual Integrations
All case studies must embed a visual placeholder styled matching the corporate crisis palette inside a hero element container:
```html
<div class="blog-hero-container">
  <!-- Interactive or graphic assets mapped by the Nano Banana Pro asset pipeline -->
</div>
```
