Product Requirements Document (PRD)BrandCheck Pro — Version 1.1 (Secure Enterprise Sandbox Blueprint)1. Executive Product VisionBrandCheck Pro is a high-security, zero-data retention "Cultural Safety Firewall" engineered for brand managers, agency leads, and creative strategists. It evaluates ad copy, campaign slogans, and brand communication against deep socio-political, religious, regional, and linguistic sensitivities before publication, specifically tuned to the nuances of the Indian market.  The primary business directive for this version is absolute industry attention and conversational authority. Success is measured by everyone in the media and advertising sectors discussing the tool's precision, security, and exceptional user experience, rather than immediate conversion or subscription numbers.Core Security Thesis: Large media agencies treat unreleased campaign copy as highly confidential intellectual property. To gain their trust, BrandCheck Pro operates on a strict Zero-Data Retention Architecture. No inputs are stored, logged, or used for training data. Content is evaluated purely in volatile memory and wiped instantly post-analysis.2. Target User Persona & Skepticism MetricsThe Profile: Senior Brand Managers, Creative Directors, and Agency Compliance Officers.  The Core Pain Point: Accidental multi-million dollar PR crises, viral boycotts (#Boycott campaigns), and brand equity damage caused by tone-deaf phrases or regional double entendres.  The Barrier to Adoption: Extreme skepticism toward generic AI wrappers, coupled with strict enterprise IT security bans on third-party tools that log internal creative assets.The Solution: An ultra-secure, terminal-style utility that values privacy as highly as analytical depth.3. Functional Requirements & Core Features4.1 "Risk Terminal Minimalism" Homepage & SandboxThe Viewport: A unified, single-page application dashboard optimized for desktop and mobile displays.Theme Adaptability: Natively maps to the user's system preferences (Light/Dark mode) upon loading, featuring a dedicated, smooth-transition theme override button.The Inputs: All fields remain empty by default with clean, vanishing placeholder typography so users never have to delete sample copy before getting started.The Security Banner: A prominent, high-fidelity data safety compliance notice permanently docked at the top of the workspace:🔒 AGENCY SECURITY ACTIVE: Zero-Data Retention Pipeline enabled. Input payloads are processed entirely in volatile RAM and deleted immediately post-analysis. No data footprints are generated or stored.4.2 Interactive Risk Metrics EngineReal-Time Token Isolation: Problematic text strings are highlighted directly using explicit, warning-state colors:Amber (Caution — Score 70-89): Ambiguous phrasing; minor refinement advised.  Crimson (High Risk — Score Below 70): High crisis probability; immediate revision strongly recommended. The MCP / API Connector Module: A prominent top-navigation node displaying structured connection guidelines, signaling to enterprise developers that the core logic can hook directly into custom internal environments.5. Data Footprint: Absolute Zero. The application does not connect to a traditional storage table for inputs. Content passes straight through TLS 1.3 encrypted streams to the model inference layer and vanishes immediately after sending the response payload back to the client.Tier Tracking Mechanics: To maintain this strict zero-data profile while enforcing usage limits, the sandbox uses a localized, low-overhead mechanism:Anonymous Tier: Restricts usage to 1 check per 24 hours per user session, tracked using a temporary, encrypted hash of the client connection data stored locally.Free Account Tier: Restricts usage to 5 checks per month. Users authenticate via a standard Google OAuth popup, writing only basic user profile parameters to the identity table, with no connection to the content being checked.Paid Tier: Unlocks unlimited access and complete risk analysis profiles.6. Engineering Strategy for AntigravityTo ensure you can execute this effortlessly inside Antigravity without deep coding experience, the system will compile into a Single-File High-Fidelity UI Blueprint (index.html) utilizing the following stack components:Utility Styles: Tailwind CSS loaded via CDN for high-end spacing, layouts, and responsive components.State Management: Alpine.js or native Vanilla JS for low-latency element transitions, theme adjustments, and token popup actions.Dual Inference Framework: Out of the box, the code embeds an intelligent, realistic simulation matching authentic deep-analysis workflows. This guarantees the prototype is instantly interactive and useful without needing immediate API setups. It also leaves a perfectly mapped integration point to drop in a free DeepSeek-V3 or Qwen 2.5 key later.
