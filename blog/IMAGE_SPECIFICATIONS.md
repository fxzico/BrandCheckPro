## Image Naming Convention

All images must follow this format for SEO:
```
brandcheckpro-[brand]-[campaign]-[descriptor]-[year].[format]
```

Examples:
- `brandcheckpro-tanishq-ekatvam-campaign-ad-2020.webp`
- `brandcheckpro-zomato-kachra-controversy-tweet-2023.webp`

## Image Technical Specifications

### Format Requirements
- **Primary:** WebP (with JPEG fallback)
- **Quality:** 85% for photos, 90% for graphics
- **Compression:** Use mozjpeg or cwebp

### Responsive Images
Provide multiple sizes for each image:
```html
<img srcset="
  image-400w.webp 400w,
  image-800w.webp 800w,
  image-1200w.webp 1200w
" sizes="(max-width: 600px) 400px, (max-width: 1000px) 800px, 1200px"
  src="image-800w.webp"
  alt="descriptive alt text"
  loading="lazy"
  decoding="async">
```

### Performance Targets
- Largest Contentful Paint image: < 100KB
- Hero images: < 150KB
- Full-width infographics: < 200KB

### Accessibility
- All images must have descriptive alt text
- Decorative images: use `alt=""` or `role="presentation"`
- Complex infographics: provide text description below

## Image SEO Checklist

- [ ] Filename includes target keywords
- [ ] Alt text is descriptive (125 characters max)
- [ ] Title attribute used sparingly (only when helpful)
- [ ] Caption provides context
- [ ] Images are responsive with srcset
- [ ] Lazy loading enabled for below-fold images
- [ ] Width and height attributes set to prevent CLS
- [ ] Structured data references images where applicable
- [ ] Image sitemap includes all blog images
- [ ] Images hosted on same domain (or CDN with proper CORS)

## Legal Considerations

### Fair Use Guidelines
- Use campaign screenshots for commentary/analysis
- Limit to necessary portions for educational purpose
- Always provide original source attribution
- Transformative use (analysis adds value)

### Trademark Notice
- Brand logos used under nominative fair use
- No endorsement implied
- Educational purpose only

### Model/Portrait Rights
- Use publicly released campaign materials only
- Avoid personal photos of individuals involved
- Focus on official brand communications
